## Install

1. Clone this repository
2. Compile this project 
3. Run this project

## Compile

Go to the root of this repository, and do:

~~~
./script/compile.sh
~~~

It makes a `map-reduce-env/` folder in your home.

## Run

Go to `~/map-reduce-env/` folder, and execute:

~~~
./script/run.sh <filename>
~~~
